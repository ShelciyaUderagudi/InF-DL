# InF-DL — In-Factory Mobility Model

Simulation of device mobility patterns within indoor factory environments 
for 6G in-X subnetwork research.

## What it does
- Models device movement patterns inside factory floor environments
- Simulates mobility scenarios relevant to ultra-dense 6G deployments
- Supports channel allocation research by providing realistic 
  device location data

## Files
- `project(mobility).ipynb` — Main mobility simulation notebook
- `mobility` — Mobility model implementation

## Tools & Libraries
- Python 3
- Jupyter Notebook

## Context
Supporting work for M.Sc research on 6G in-X subnetworks at the 
University of Bremen.
